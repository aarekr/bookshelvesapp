## Project Requirements

Requirement                       | Items                                      |
----------------------------------|--------------------------------------------|
Purpose                           | To manage books on reading list, reading started and completed books |
Responsiveness                    | Layouts/breakpoints: mobile, tablet, large screen |
Maximum width                     | BoxConstraints maxWidth is 800 |
Form                              | Form for entering books to reading list |
Application remembers             | All added books and their statuses |
Interaction                       | Books can be marked reading started, completed and deleted |
Statistics                        | Number of books not read, reading, completed |
Main + min. 3 screens             | Home, AddBook, NotStarted, Reading, Completed, Language |
Min. 1 screen uses path variables | Language Screen uses path variables and has no bottom navigation buttons |
App online                        | [Application on GitHub Pages](https://aarekr.github.io/bookshelvesapp-demo/) |
